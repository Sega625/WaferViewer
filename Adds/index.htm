Введение

Программа WaferViewer предназначена для просмотра и анализа файлов статистики с расширение .sts и .mdb

С ее помощью можно:

  •  просмотреть карту годности

  •  просмотреть измеренные параметры каждого кристалла

  •  просмотреть разбраковку по тестам

  •  просмотреть результаты анализа на ПД (повторяющийся дефект)

  •  распечатать все данные по пластине на принтер

Просмотр карты годности

Для просмотра карты годности пластины нужно сначала загрузить ее в программу. Это можно сделать 3-мя способами:

  1. Нажать на пункт меню "Файл", затем в ниспадающем меню нажать "Открыть" (либо нажать сочетание клавиш "Ctrl+O") и выбрать нужный файл статистики.

  2. Перетащить мышкой файл статистики в окно программы.

  3. Два раза кликнуть на файл статистики (только для файлов .sts). При этом программа должна быть перед этим запущена хоть раз.

После этого у вас должно появиться похожее окно:

В панели "Инф-ция о пластине" выводится основная информация по пластине. Также в этой панели вы можете изменить основные цвета отображения кристаллов, нажав на соответствующий цветной значок.

В панели "Настройки" вы сможете, используя кнопки виртуального джойстика уменьшить или увеличить размеры и масштаб карты годности. Также можно менять масштаб карты с помощью колесика мыши и одновременно нажатой клавишей "Ctrl". Кнопка "А" предназначена для автоматического выбора масштаба карты годности. При достаточном размере кристаллов на их изображениях появятся цифры, обозначающие номер брака. Кнопка "Б" подсвечивает базовый кристалл. Кнопка "П" делает прозрачной область карты годности и панель "Инф-ция о пластине", ползунок плавно меняет прозрачность всего окна (для WinXP и выше). Это может быть полезным, например, при сравнении 1 и 2 замеров.

Просмотр параметров по каждому кристаллу

Для просмотра параметров кристалла нужно в области отображения карты годности нажать на нужный кристалл, при этом появится окошко со всеми измеренными параметрами для этого кристалла, а сам кристалл подсветится желтым цветом:

Просмотр разбраковки по тестам

Для просмотра разбраковки по тестам нужно нажать на пункт меню "Анализ", затем в ниспадающем меню нажать "Разбраковка" или нажать сочетание клавиш "Ctrl+R" (при этом пластина уже должна быть загружена). Появится следующее окно:

Отображаются только браки, количество которых больше нуля. При желании можно изменить цвет любого брака и немедленно увидеть на карте его распределение нажав на соответствующий цветной значок:

Просмотр ПД

Для просмотра ПД нужно нажать на пункт меню "Анализ", затем в ниспадающем меню нажать "ПД" или нажать сочетание клавиш "Ctrl+D" (при этом пластина уже должна быть загружена и иметь ненулевой размер кадра). Появится следующее окно:

Цифры в клеточках указывают сколько годных кристаллов в данной позиции кадра. Если количество годных равно 0, то кристалл окрашивается в красный цвет (ПД), в противном случае - в зеленый.

в начало

Распечатка всех данных по пластине на принтер

Для распечатки всех данных по пластине на принтер нужно нажать на пункт меню "Файл", затем в ниспадающем меню нажать "Распечатать" (либо нажать сочетание клавиш "Ctrl+P") и в стандартном диалоге выбора принтера нажать "OK".
